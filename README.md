## Requisitos tener instalado Docker y Docker compose
## Para iniciar solo usar
```
docker-compose up -d
```

### comandos de utilidad

## para detener

```
docker-compose stop
```

## para najar las imagenes y contenedores
```
docker-compose down
```

### Borra todo el contenido de lo cargado
```
docker-compose down --rmi all
```
